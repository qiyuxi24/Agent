## Code Server 方案切换：exe → Node.js npm 包（2026-07-10 晚上 19:20）

### 根本原因
code-server v4.127.0（最新版）**不再发布 Windows 独立 .exe**！GitHub Releases 只有 `package.tar.gz`（npm 包）。之前用 `latest/download/code-server-windows-amd64.exe` 链接返回 404。

### 新方案
从 `package.tar.gz` 入手：解压 → `npm install --production --ignore-scripts` → 用 `node out/node/entry.js` 启动。

2. `curl -k -L` 下载成功（走 Clash 代理，绕过 TLS 证书问题）
3. `tar -xzf` 解压 + `npm install --production --ignore-scripts`（依赖安装约 2 分钟）
4. `node out/node/entry.js --version` → **code-server 4.127.0** ✅
5. 总大小：**211.9 MB**（含 node_modules）
6. 目录重命名：`binaries/release/` → `binaries/code-server/release/`

### 代码变更（第二轮重构）
